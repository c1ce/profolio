# profolio

개인 포트폴리오를 다른 사용자들에게 오픈하고 기업의 인사담당자가 해당 포트폴리오를 열람한 뒤, 기업과 회원간의 Contact 할 수 있는 종합 포트폴리오 관리 시스템

Key Feature.
--
1. 사용자 계정을 개인/기업으로 분리
2. 포트폴리오 CURD
3. 포트폴리오 탐색 기능
4. 개인 사용자와 기업간의 contact 기능
